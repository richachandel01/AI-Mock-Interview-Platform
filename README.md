# AI Mock Interview Platform

## Project Overview

AI Mock Interview Platform is a full-stack SaaS application that helps students and job seekers practice technical interviews using AI-generated questions, AI evaluation, voice-based answers, analytics, and downloadable reports.

---

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Axios
* Framer Motion
* Recharts

### Backend

* Java Spring Boot
* Spring Security
* JWT Authentication
* Hibernate / JPA
* Maven

### Database

* PostgreSQL

### AI Integration

* Gemini API
* OpenAI (Fallback)
