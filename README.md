# AI Mock Interview Platform

## Project Overview

AI Mock Interview Platform is a full-stack SaaS application designed to help students and job seekers prepare for technical interviews using AI-powered mock interview sessions.

Users can:

* Register and login securely
* Select interview roles
* Practice mock interviews
* Receive AI-generated feedback
* Track interview history
* Download interview reports
* Analyze performance through dashboards
